Basic Working Instructions:

  • First we run the Makefile using make and the run the executable ./assign1.

  • We are in the Modelling mode at the start. We can draw polygons by doing left click on the screen. For removing points use shift + left click.

  • We can save the model in a .raw format by pressing K. The points are stored in X,Y,Z,R,G,B format.

  • Now we can go to the Inspection mode by pressing I and then use keys Up Arrow and Down Arrow to rotate about the X-axis, Left Arrow and Right Arrow to rotate about the Y-axis, PgUp and PgDn to rotate about the Z-axis,

  • We can also translate the object in the Inspection mode using keys W, S, A, D, Z, X to translate in +ve or -ve Y, X and Z axis respectively.

  • To load a file press L and then enter filename (with the format). This is done in Inspection mode.

  • The function centroid in the file centroidFunctions.cpp finds the centroid of the object. On pressing R the centroid of the object is shifted to the origin.

  • We can go back to the Modelling view by pressing M and then can start drawing new models directly.
